## Гид по технике ФСТмедиа (Telegram-бот)

[![Run tests](https://github.com/yanachkj-png/Gid-po-tehnike-FST/actions/workflows/test.yml/badge.svg)](https://github.com/yanachkj-png/Gid-po-tehnike-FST/actions)

Telegram-бот для студенческого СМИ ФСТмедиа. Позволяет оперативно получать инструкции по работе с техникой – камеры, радиосистемы, съёмочный комплект. Вся информация структурирована, сопровождается фотографиями и пошаговыми пояснениями.

---

### 📌 Функционал

- Главное меню с тремя разделами: Съёмочный комплект, Радиосистемы, Камеры.
- Вложенные подразделы с детальными гайдами.
- Отправка фотографий (включая альбомы с несколькими снимками).
- Умная навигация:
  - Кнопки «Далее» / «Назад» для пошагового просмотра.
  - Кнопка «В меню раздела» для возврата без потери контекста.
  - Кнопка «К разделам» для выхода в главное меню.
- Автоматическое удаление старых сообщений – чат всегда чист.
- Защита от спама (ограничение частоты сообщений).
- Дружелюбная обработка случайного текста – бот подсказывает, как пользоваться кнопками.

---

### 📖 Инструкция по использованию

#### 1. Начало работы  
Откройте Telegram и найдите бота по ссылке [@fstguide_bot](https://t.me/fstguide_bot) или введите его имя в поиске. Нажмите «Старт» или отправьте команду `/start`.

#### 2. Навигация по разделам  
Главное меню содержит три кнопки. Выберите нужный раздел:
- **Съёмочный комплект** – как правильно собрать и переносить оборудование.
- **Радиосистемы** – подключение, использование и зарядка радиомикрофонов Sennheiser.
- **Камеры** – работа с камерой Sony FDR-AX53, картами памяти, аккумуляторами и штативом.

Внутри каждого раздела доступны подразделы с конкретными инструкциями. Перемещайтесь по шагам с помощью кнопок «Далее» / «Назад». В любой момент можно вернуться в меню раздела или в главное меню.

#### 3. Получение информации  
Каждый шаг сопровождается текстовым пояснением и, при необходимости, фотографией. Если в инструкции несколько фото, они отправляются альбомом.

#### 4. Что делать, если бот не отвечает  
Если вы отправили текст вместо нажатия на кнопку, бот покажет подсказку с доступными командами. Также всегда работает команда `/start` для перезапуска.
