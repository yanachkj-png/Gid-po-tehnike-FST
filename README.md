# Гид по технике ФСТмедиа — Telegram-бот

[![Run tests](https://github.com/yanachkj-png/Gid-po-tehnike-FST/actions/workflows/test.yml/badge.svg)](https://github.com/yanachkj-png/Gid-po-tehnike-FST/actions)

## О проекте

Telegram-бот для студенческого СМИ ФСТмедиа, который помогает студентам и сотрудникам быстро находить инструкции по работе с техникой.

Вся информация структурирована, снабжена фотографиями и пошаговыми пояснениями.

## Функциональность

- Главное меню с тремя разделами:
  - Съёмочный комплект
  - Радиосистемы
  - Камеры
- Вложенные подразделы с подробными инструкциями
- Отправка фотографий (альбомы и одиночные)
- Умная навигация:
  - Кнопки Далее / Назад для пошагового просмотра
  - Кнопка В меню раздела для возврата в текущий раздел без потери контекста
  - Кнопка К разделам для выхода в главное меню
- Защита от спама (ограничение частоты сообщений)
- Автоматическое удаление старых сообщений для чистоты чата
- Дружелюбная обработка случайного текста: бот подсказывает, как пользоваться кнопками

## Технологии

- Python 3.10+
- aiogram 3.x — современный фреймворк для Telegram-ботов
- aiohttp — веб-сервер для поддержки работы на Render
- pytest + pytest-asyncio — юнит-тесты для ключевых функций
- GitHub Actions — автоматический запуск тестов при каждом пуше

## Деплой

Бот развёрнут на Render.com (бесплатный тариф) и работает 24/7 благодаря:
- встроенному веб-серверу (порт 8080)
- мониторингу через UptimeRobot (пинг каждые 5 минут)

## Локальный запуск

### 1. Клонируйте репозиторий

git clone https://github.com/yanachkj-png/Gid-po-tehnike-FST.git
cd Gid-po-tehnike-FST

### 2. Установите зависимости

pip install -r requirements.txt

### 3. Настройте переменную окружения

Создайте файл .env и добавьте:
BOT_TOKEN=ваш_токен_от_BotFather

### 4. Запустите бота

python bot.py

## Тесты

Проект покрыт юнит-тестами (проверка клавиатур, отправки сообщений, удаления, состояний).

Запуск тестов:
pytest tests/

Тесты автоматически запускаются в GitHub Actions при каждом пуше в ветку main.
